Each of the cities in the dataset.

``` r
city_results <- 
  homicide_data %>%
  group_by(city_state) %>%
  summarize(
    total = n(),
    unsolved = sum(disposition %in% c("Closed without arrest", "Open/No arrest")),
    .groups = "drop"
  ) %>%
  mutate(
    prop_test = map2(unsolved, total, ~prop.test(.x, .y))
  ) %>%
  mutate(
    prop_tidy = map(prop_test, tidy)
  ) %>%
  unnest(prop_tidy) %>%
  select(
    city_state,
    estimate,
    conf.low,
    conf.high
  )
```
